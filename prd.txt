PRODUCT REQUIREMENTS DOCUMENT
FEATURE: Fullscreen Loader Animation

OBJECTIVE
Implement a fullscreen loader animation similar to the provided reference.

The loader should display a percentage counter, a vertical progress indicator line, and the brand name in the center.

When the loader reaches 100%, the loader should animate upward to reveal the website underneath.

The animation must feel cinematic and smooth, similar to Awwwards-style website loaders.

Tech Stack
React
TailwindCSS
GSAP

------------------------------------------------

LOADER LAYOUT

The loader should cover the entire viewport.

Container properties:

position: fixed
top: 0
left: 0
width: 100vw
height: 100vh
background: #000000
z-index: 9999

All loader elements should be positioned inside this container.

------------------------------------------------

CENTER BRAND TEXT

Display the website brand name in the center of the loader.

Brand name:

"SYNTH AUDIO"

Typography style:

font-family: Space Grotesk or similar modern sans-serif
font-weight: 600
font-size: 42px
letter-spacing: 0.12em
text-transform: uppercase
color: white

Position:

center horizontally
center vertically

Goal:
The loader should feel like a premium brand introduction.

------------------------------------------------

PERCENTAGE COUNTER

Display a large loading percentage in the bottom-left corner.

Example:

59%

Typography:

font-family: Bebas Neue or similar bold display font
font-size: 120px
font-weight: 900
color: white

The "%" symbol should be smaller:

font-size: 36px

Position:

bottom: 60px
left: 80px

Goal:
Create a bold visual indicator of loading progress.

------------------------------------------------

VERTICAL PROGRESS LINE

Add a vertical orange progress line on the left side of the screen.

Style:

position: absolute
left: 40px
bottom: 0
width: 6px
height: dynamic
background: #ff4d2e

The height of this line should increase as the percentage increases.

Example mapping:

0% → height: 0vh
100% → height: 100vh

Goal:
Create a visual indicator that tracks the loading progress.

------------------------------------------------

LOADING ANIMATION

Simulate loading progress using GSAP.

Animation behavior:

Start at 0%
Increment gradually until 100%

Animation duration:

2.5 seconds total

Counter animation:

Update the percentage text dynamically.

Example:

0 → 100

The vertical line height should update in sync with the percentage.

------------------------------------------------

LOADER EXIT ANIMATION

When the percentage reaches 100:

Wait 0.4 seconds.

Then animate the loader container upward.

Animation:

translateY: 0 → -100vh
duration: 1s
ease: power3.inOut

This should reveal the website underneath.

------------------------------------------------

WEBSITE REVEAL

Before the loader exits, the website content should already be rendered underneath.

As the loader moves up, the website becomes visible.

Optional enhancement:

Add slight fade-in to the main hero section.

opacity: 0 → 1
duration: 0.6s

------------------------------------------------

SMOOTHNESS REQUIREMENTS

All animations must use GPU-friendly properties.

Use:

transform
opacity

Avoid layout reflows.

------------------------------------------------

VISUAL STYLE GOAL

The loader should feel:

minimal
cinematic
premium
brand-focused

Inspired by Awwwards website loaders.

------------------------------------------------

EXPECTED RESULT

When the website loads:

1. Black fullscreen loader appears
2. Brand name sits centered
3. Large percentage counter increases
4. Orange vertical line grows upward
5. At 100%, the loader slides up
6. The website hero section is revealed smoothly

------------------------------------------------

SECTION — BRAND TRANSITION TO NAVBAR

OBJECTIVE

The brand name displayed in the center of the loader should animate into the navbar logo position when the loader finishes.

This creates a continuous brand transition and avoids the logo appearing twice.

------------------------------------------------

INITIAL STATE

During the loader:

The brand name is centered.

Position:

center horizontally
center vertically

Typography:

font-family: Space Grotesk
font-size: 42px
font-weight: 600
letter-spacing: 0.12em
color: white

------------------------------------------------

FINAL STATE

The brand should move to the navbar logo position.

Navbar logo position:

top: 28px
left: 80px

The font size should shrink slightly.

Final size:

font-size: 20px

------------------------------------------------

ANIMATION SEQUENCE

When the loading counter reaches 100%:

Step 1 — Pause briefly

delay: 0.3s

Step 2 — Animate the brand text

Move from center position to navbar position.

Animation properties:

translateX
translateY
scale: 1 → 0.45

duration: 0.9s
ease: power3.inOut

The text should smoothly travel to the navbar location.

------------------------------------------------

NAVBAR SYNC

While the brand is moving:

The navbar should fade in.

Animation:

opacity: 0 → 1
duration: 0.5s
delay: 0.3s

When the brand reaches the navbar position:

The loader brand element becomes the navbar logo.

------------------------------------------------

LOADER EXIT

After the brand reaches the navbar position:

Animate the loader panel upward.

translateY: 0 → -100vh

duration: 1s
ease: power3.inOut

The website becomes visible underneath.

------------------------------------------------

EXPECTED RESULT

The loading sequence should feel like:

1. Loader appears
2. Counter increases
3. Vertical progress line grows
4. Brand sits centered
5. At 100% brand moves to navbar
6. Loader slides up revealing the site
